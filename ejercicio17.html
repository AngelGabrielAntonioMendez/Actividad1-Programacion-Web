Conceptos Clave de JavaScript

Ámbito de Variables (Scope)

Determina la visibilidad o accesibilidad de las variables. En este ejercicio:

  • Scope Global: La instancia del gestor const gestor = crearGestorTareas(); y la variable de versión son globales.
  • Scope Local: La variable let tareas dentro de crearGestorTareas() solo existe en su interior y es inaccesible desde la consola del navegador directamente. 
// Ejemplo en código:
let version = "1.0"; // Global

function test() {
    let local = "Privado"; // Local
}

Closures (Clausuras)

Un closure es una función que "recuerda" su entorno léxico original. Permite la encapsulación y creación de variables privadas:

  • El objeto retornado por crearGestorTareas() contiene las funciones agregarTarea() y eliminarTarea().
  • Estas funciones mantienen una referencia activa a la variable local tareas incluso después de que la función externa haya finalizado. 
// Creando el closure:
const gestor = crearGestorTareas();
// 'tareas' se mantiene vivo y seguro en memoria
gestor.agregarTarea("Estudiar");

Local Storage y JSON

Permite la persistencia de datos en el navegador de manera local y permanente (incluso al cerrar la pestaña):

  • JSON.stringify(): Convierte el array de objetos de JavaScript a formato texto JSON para almacenarlo.
  • JSON.parse(): Recupera el texto JSON y lo convierte de vuelta a un objeto/array de JavaScript al recargar la página.
// Guardar en Storage:
localStorage.setItem('tareas', JSON.stringify(tareas));

// Recuperar del Storage:
const data = JSON.parse(localStorage.getItem('tareas'));
